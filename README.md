--프로젝트 소개--

이 프로젝트는 사용자가 자신의 성격 유형(MBTI)을 검사하고, 결과를 저장 및 관리할 수 있는 간단한 웹 애플리케이션입니다. 로그인 상태에 따라 개인화된 기능을 제공하며, JSON Server를 사용하여 데이터를 관리합니다.
